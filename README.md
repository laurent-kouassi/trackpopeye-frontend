

# TRAPPOPEYE-FRONTEND

## Features

- ReactJS
- REST API
- BoostrapUI
- Leaflet Map

## Requirements

- [node & npm](https://nodejs.org/en/)

## Installation

- `git clone https://github.com/laurent-kouassi/trackpopeye-frontend.git`
- `cd trackpopeye-frontend`
- `npm install`
- `npm start`

### Server

- start the server
